## 게임 개요

MadRush<br/><br/>
길을 따라 이동하는 상대편 병사들이 성문을 통과하지 못하도록 막는 게임입니다.<br/>
궁수, 병영, 마법사 타워를 길목에 설치해 상대편 병사들을 공격할 수 있습니다.<br/>
